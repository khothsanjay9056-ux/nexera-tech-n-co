# This file explains what to do since I can't create images

## What is an OG Image?
The image that appears when you share your website on social media (Facebook, LinkedIn, Twitter, etc.)

## Specifications
- **Size**: 1200px width × 630px height
- **Format**: JPG or PNG
- **File name**: og-image.jpg
- **Location**: Save to `public/og-image.jpg`

## What to Include
1. **Your Logo** (from public/logo.png)
2. **Company Name**: Nexera Technology and Consultancy
3. **Tagline**: "Stop Guessing. Start Growing."
4. **Background**: Use one of your gradient backgrounds
   - Option 1: Purple gradient (#667eea → #764ba2)
   - Option 2: Blue gradient (#4facfe → #00f2fe)
5. **Optional**: Icons representing your services (charts, shopping cart, automation)

## How to Create
**Option 1: Use Canva (Free)**
1. Go to canva.com
2. Create custom size: 1200 x 630
3. Add your logo, text, and gradient background
4. Download as JPG
5. Save as `public/og-image.jpg`

**Option 2: Use Figma (Free)**
1. Create 1200x630 frame
2. Design with your branding
3. Export as JPG

**Option 3: Hire on Fiverr**
- Search "OG image design"
- Cost: $5-20
- Provide your logo and brand colors

## Quick Template
If you want to skip design for now, use a simple text-based OG image:
- Solid gradient background
